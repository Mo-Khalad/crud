<h1>CRUD</h1>

<p>The Curd app lets you store, edit, delete, and search product data. It retains data when the page is refreshed via local storage and features a data validation system built using HTML, CSS, Bootstrap, and JavaScript.
</p>

<h3>tools</h3>

<h5>html  ,  css ,  bootstrap  and js </h5>



<h3>features</h3>
<ul>
  <li>Add a product</li>
  <li>Delete a product</li>
  <li>Edit product data</li>
  <li>Search for a product by name</li>
  <li>Save data on refresh via local storage</li>
  <li>Validation is performed using regex</li>
</ul>
